INSTRUCTIONS FOR USE:
====================
1. Print this template or copy to a text editor
2. Open each COP document (start with FY24 ACFR.pdf)
3. Fill in as much information as you can find
4. Don't worry if some fields are blank - we'll work with what's available
5. Focus on the core financial data first: amounts, rates, dates
6. Save each completed template with the COP name (e.g., "EA801455_extracted.txt")
7. When done with all documents, fill in the aggregate summary
8. Send me the completed templates and I'll convert to JSON format
